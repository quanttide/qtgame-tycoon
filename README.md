# qtgame-tycoon

量潮科技模拟器 — 以 CEO 视角经营一家产教融合的科技公司。

管理项目、建设团队、构建制度、完成蜕变。

## 快速开始

直接用浏览器打开 `src/index.html` 即可游玩。

## 玩法概要

- 承接数据与教育类项目，积累资金与知识
- 招聘和培养团队成员
- 投资战略升级（工具链、课程体系、平台、品牌、自动化）
- 触发随机事件，做出经营决策
- 激活"产教融合循环"，获得效率加成
- 每季度末结算评分，持续扩张

## 项目结构

```
src/index.html     # 游戏入口（单页 HTML）
doc/               # 文档
.github/           # CI/CD
```

## 技术栈

纯前端，单页 HTML + CSS + JavaScript，无外部依赖。
