## 二、项目作战图（核心区）

### 2.1 项目管道概览
- **可承接项目** — 来自市场 / 客户推荐的待接项目列表
- **执行中项目** — 每个项目的进度条、客户、类型（数据/教育）、预计结项时间
- **本周完成** — 最近完成的项目及收益
- **延期风险** — 进度落后于预期的项目（红色预警）

### 2.2 项目卡片信息
每个项目显示：客户名称、项目类型标签（📊数据 / 📚教育）、合同金额、工期（周）、当前进度%

## 三、团队与人才面板

### 3.1 团队构成
- 成员列表，每人显示：姓名、数据能力值、教育能力值、当前状态（空闲/执行中）
- 技能分布雷达：数据 vs 教育的能力加权分布
- 人均产出：最近季度的人均创收

### 3.2 实习生管道
- 当前实习生数量
- 实习生来源（合作高校）
- 实习生转化率（转为正式成员的概率）

### 3.3 人才缺口
- 当前项目需求 vs 团队产能的对比
- 提示是否需要招聘

## 四、产教融合循环（核心机制仪表）

这是量潮科技最独特的能力——数据项目吸引学生，学生支撑项目交付，循环增强。

| 维度 | 指标 |
|---|---|
| 📊 数据知识 | 通过数据项目积累的知识点数 |
| 📚 教育知识 | 通过教育项目积累的知识点数 |
| 🔄 融合度 | Lv.0 ~ Lv.5，等级越高，效率加成越大 |
| 🔗 循环状态 | 描述文字（未激活 → 初显成效 → 良性运转 → 高度融合） |

### 融合度升级条件
- 数据知识和教育知识同时达到阈值（每 20 点升一级）
- 融合度提供：项目单价加成 +10%/级、全局效率加成

## 信息架构设计原则

1. **一眼可见**：最关键的 6 个数字必须在 header 常驻
2. **层次分明**：核心区 → 项目/团队 → 战略投资 → 折叠面板
3. **红绿预警**：用颜色标记异常状态（资金紧张、项目延期、人才缺口）
4. **即时反馈**：每完成一个动作，相关指标立即刷新
5. **移动优先**：适配手机竖屏，单手可操作
